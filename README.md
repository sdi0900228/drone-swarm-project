# AI-Enabled Drone Swarm Mini Project

## Project Goal
This project aims to implement a small end-to-end inter-drone information exchange pipeline.

The system uses:
- YOLO for object detection
- MQTT for message exchange
- simple rule-based logic for decision support

The goal is not to build a full large-scale drone swarm simulation, but to integrate heterogeneous software components into a small working collaborative scenario.

## Planned Modules
1. Object detection
2. Message exchange
3. Decision logic
4. End-to-end integration
